# Databaza-Banka-NLB
Ky projekt i bazës së të dhënave Banka NLB  fokusohet në hartimin dhe zbatimin e një  Sistemi të Menaxhimit të Bankes për të rritur efikasitetin, sigurinë dhe funksionalitetin e  operacioneve bankare. Ku duke përfshirë gjashtë entitete thelbësore: Klientin, Llogarinë,  Transaksionin, Kredin, Punonjësin dhe Degën.
